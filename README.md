## My process

1. Downloaded the starter files
2. Downloaded the figma files, and opened the figma project to obtain exact color shades and text/content sizes
3. Configured the external stylesheet
4. Created a parent container, followed by a container for the QR code element
5. Created smaller div containers for the QR code image as well as the text
6. Added CSS properties according to the figma information

### Built with

-   CSS Grid
